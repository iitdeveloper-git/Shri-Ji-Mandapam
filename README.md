## 🏰 About the Project

This repository contains the official website of **Shri Ji Mandapam**, a premium wedding lawn and air-conditioned banquet hall located near Manona Dham in Aonla, Uttar Pradesh.

The experience is designed for families planning weddings, receptions, engagements, sangeet ceremonies and other memorable celebrations. It combines elegant visuals, bilingual content, smooth animations and a simple WhatsApp-powered enquiry flow.

> **श्री जी मंडपम** — भव्य आयोजन, शानदार मेहमाननवाज़ी और यादगार पलों के लिए एक खूबसूरत स्थान।

## 🚀 Website Features

- Fully responsive design for mobile, tablet and desktop
- Immersive hero sections and curated venue galleries
- Smooth page transitions powered by Framer Motion and GSAP
- Venue, room, dining and nearby-attraction information
- Dynamic gallery categories and blog detail pages
- Booking form with direct WhatsApp enquiry support
- English and Hindi content for better local accessibility
- SEO metadata, Open Graph cards, sitemap and robots configuration
- Structured data for improved search-engine discoverability
- Reusable, type-safe React components

## 🗺️ Main Routes

| Route | Purpose |
| --- | --- |
| `/` | Home and venue overview |
| `/about-us` | Story, facilities and contact information |
| `/venues-services` | Venues, services and celebration options |
| `/gallery` | Curated photo gallery |
| `/gallery/[category]` | Category-specific gallery pages |
| `/blog` | Hospitality and travel articles |
| `/blog/[slug]` | Individual blog articles |
| `/book-now` | Room and enquiry booking experience |
| `/[slug]` | Dynamic service and information pages |

## 🛠️ Built With

- **Framework:** Next.js 15 with App Router
- **UI:** React 19, TypeScript and Tailwind CSS
- **Motion:** Framer Motion, GSAP, Lenis and Swiper
- **Forms:** React Hook Form and Zod
- **Icons:** Lucide React
- **SEO:** Next.js Metadata API, sitemap and JSON-LD schema

## 💻 Getting Started

### Prerequisites

- Node.js **20 LTS** recommended (minimum supported version: 18.18)
- npm 9 or newer

### Installation

```bash
git clone https://github.com/iitdeveloper-git/Shri-Ji-Mandapam.git
cd Shri-Ji-Mandapam
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```
