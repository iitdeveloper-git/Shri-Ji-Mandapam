# 🎉 Shri Ji Mandapam – Premium Banquet Hall & Wedding Lawn

![Banner](https://raw.githubusercontent.com/iitdeveloper-git/Shri-Ji-Mandapam/main/public/shri_ji_facade_day.jpg)

Welcome to the **official website** of **Shri Ji Mandapam**, a premier banquet hall and wedding lawn located in Aonla, Uttar Pradesh. This repo contains the full **Next.js** application that powers the stunning, responsive, and highly interactive web experience for our clients.

---

## ✨ Features

- **Dynamic Hero Sections** with gorgeous background images and smooth gradients.
- **Responsive Design** – desktop, tablet, aur mobile devices par bina kisi dikkat ke kaam karta hai.
- **Our Venues & Services** page par dikhaya gaya hai:
  - AC Banquet Hall (800 seats) 🍽️
  - 15,000 sq ft Garden Lawn 🌿
  - 10 Luxury Guest Rooms 🛏️
  - Premium vegetarian catering, power backup, décor support, aur bhi bahut kuch.
- **Book Now** experience Booking.com jaise, powerful filters aur instant WhatsApp inquiry modal ke saath.
- **Google Map** integration clean UI ke saath (extra text ya shortcuts nahi dikhte).
- **Contact Footer** mein phone, email, address aur 24‑hour support diya gaya hai.
- **SEO‑ready** – proper title tags, meta descriptions, heading hierarchy, aur semantic HTML.
